# CES Evolution System

Self-evolving artificial intelligence — open source and free forever.

**Live demo:** https://ces-evolution-system.netlify.app/

Built with: React, TypeScript, Three.js, GSAP, Recharts, Zustand

## Getting Started

```bash
npm install
npm run dev
```

## Features

- Genetic algorithm evolution engine
- Real-time 3D particle visualisation
- Intelligence panel (plain English learning display)
- CSV upload for real data
- Export results as PDF/CSV/JSON/PNG
